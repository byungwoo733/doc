S.no. Environment vars Description
1 VE_LD_LIBRARY_PATH Environment variable pointing to directories where the dynamic loader should look for shared libraries.
VE_LD_LIBRARY_PATH on VE is similar to LD_LIBRARY_PATH in x86_64
2 VE_LD_PRELOAD This environment variable sets the preloading path for dynamic linker (ld) and that allows to load a user specified, shared library before all other shared libraries which are linked to an executable get loaded.
VE_LD_PRELOAD on VE is similar to LD_PRELOAD in x86_64
