S.no. 環境カンキョウ変数ヘンスウ 定義テイギ
1 VE_LD_LIBRARY_PATH 動的ドウテキリンカが共有キョウユウライブラリを探サガすディレクトリを指定シテイする環境カンキョウ変数ヘンスウ。
VE の VE_LD_LIBRARY_PATH は x86_64 における LD_LIBRARY_PATH 相当の環境変数です。
2 VE_LD_PRELOAD この環境カンキョウ変数ヘンスウは動的ドウテキリンカにプリロードする共有キョウユウライブラリのパスを設定セッテイします。実行ジッコウファイルにロードされるどの共有キョウユウライブラリよりも先サキに、ユーザが指定シテイするライブラリをロードする事コトが可能カノウです。
VE の VE_LD_PRELOAD は x86_64 における LD_PRELOAD 相当の環境変数です。
